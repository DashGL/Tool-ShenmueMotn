# Tool-ShenmueMotn
Tools for parsing motn files from Shenmue with Nodejs and C

- Contains tools in C and Nodejs
- Instructions on how to run each one

# Motion (.motn) Format

This section covers technical information about the .motn format used in Shenmue
- FK animations
- F-curves
- Broken down into "blocks"

## Name Table

- List of names + offset 


## Header

- Length of each block

## Block 1

- Bone Id + axis

## Block 2

- Number of key frames

## Block 3

- Specific Key frames

## Block 4

- Easing Values

## Block 5

- Float values for easing
- FLoat value for key frame
